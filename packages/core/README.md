# @kamox/core

Core library for KamoX - Web Extension development server.

This package provides the base classes and utilities for building KamoX adapters.

## Installation

```bash
npm install @kamox/core
```

## Usage

This is a library package used by KamoX plugins. For end-user documentation, please see the [kamox](https://www.npmjs.com/package/kamox) package.

## Documentation

For full documentation, please visit the [KamoX GitHub repository](https://github.com/yourorg/kamox).
