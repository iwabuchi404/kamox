# kamox

KamoX CLI - AI-friendly development server for Web Extensions.

## Installation

```bash
npm install -g kamox
```

## Usage

```bash
# Auto-detect project configuration
kamox detect

# Start Chrome Extension development server
kamox chrome --auto-build
```
